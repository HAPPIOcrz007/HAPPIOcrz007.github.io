# =(#1866E1)[link ➽=](https://github.com/HAPPIOcrz007/4-x-4-Registor-RWC)

A custom =(#f0a030)PCB= implementing a =(#2da44e)4×4 memory register array= entirely from =(#ff6b6b)74-series TTL ICs= — no microcontrollers, no FPGAs. Pure combinational and sequential logic.

## =(#f0a030)Design=

- =(#2da44e)PIPO (Parallel In, Parallel Out)= architecture for the register array
- =(#ff6b6b)Synchronous read/write= with =(#58a6ff)asynchronous clear= — designed to mirror how real register files behave
- =(#f0a030)74-series TTL= chosen deliberately: every gate, every flip-flop is visible and traceable on the board

## =(#58a6ff)Physical Interface=

Engineered for hands-on learnability: input =(#f0a030)DIP switches=, output =(#2da44e)LEDs=, and clearly labelled control lines so students can probe the circuit and understand exactly what's happening at each clock edge.

- =(#2da44e)★ Finalist= at the DOEEE Hardware Hackathon, MITWPU
- =(#58a6ff)★ Adopted by the college= as a recommended learning kit for junior-year students in digital electronics
